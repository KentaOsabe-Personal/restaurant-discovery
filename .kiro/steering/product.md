# プロダクト概要

レストラン発見・探索プラットフォーム。ユーザーが新しい飲食店を見つけ、情報を整理・共有できるWebアプリケーション。

## コア機能

- **自然文レストラン検索**（実装済み）: 「渋谷でリーズナブルなイタリアン」のような自然文クエリで店舗を探せる
- **AIレコメンデーション**（実装済み）: OpenAIが候補店から最適な3〜5件を選定し、推薦理由を提示
- ユーザーによるレビュー・評価（予定）

## AI検索パイプライン

```
自然文クエリ
  → QueryParserService（OpenAI: エリア/ジャンル/価格帯を抽出）
  → GooglePlacesService（Google Places API: 候補店を取得）
  → RecommendationService（OpenAI: 最適な店を推薦・理由付け）
  → フロントエンド表示
```

## 主要ユースケース

- 自然文で条件（エリア・ジャンル・価格帯）を指定してレストランを探す
- AIが選んだおすすめ店と推薦理由を確認して店を選ぶ
- 行きたい店・行った店を記録・管理する（予定）

## 価値提案

シンプルで直感的なUIで、レストラン探しの体験を向上させる。フロントエンドとバックエンドを疎結合に保ち、将来的な機能拡張に対応しやすい設計。
