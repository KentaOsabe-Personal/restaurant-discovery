# 実装計画

- [x] 1. 食べログ検索 URL を生成する純粋関数を実装する
  - 店名を受け取り食べログ新潟の検索 URL（`https://tabelog.com/niigata/rstLst/?vs=1&sk={エンコード済み店名}`）を返す関数を追加する
  - `name.trim()` が空文字の場合は `null` を返し、リンク非表示の条件分岐に利用できるようにする
  - 日本語・スペース・記号を `encodeURIComponent` でパーセントエンコードし、URL として安全な形式にする
  - 新潟県固定パス（`/niigata/`）を URL に組み込み、検索精度を向上させる
  - 既存の `formatPriceLevel` と同じ配置パターンでモジュールスコープに非公開関数として定義する（`export` なし）
  - _Requirements: 2.1, 2.2, 2.3, 2.4_

- [x] 2. PlaceCard のリンクエリアを横並びレイアウトに更新し、食べログリンクを追加する

  - 既存の `block mt-2` クラスをコンテナ div 側に移動し、`<a>` 側からは除去してレイアウトを整合させる
  - URL 生成関数の返却値が `null` の場合は食べログリンクを DOM に出力しない（条件付きレンダリング）
  - リンクテキストを「食べログで見る」とし、`text-orange-500` で Google Maps リンク（`text-blue-600`）と視覚的に区別する
  - `target="_blank"` と `rel="noopener noreferrer"` を付与し、新タブ遷移とタブナビゲーション攻撃の防止を両立させる
  - _Requirements: 1.1, 1.2, 1.3, 3.1, 3.2, 3.3_

- [x] 3. PlaceCard のテストを追加する
- [x] 3.1 食べログリンクの表示・非表示の基本ケースを検証する

  - 空白のみの店名の場合に食べログリンクが表示されないことを確認する

  - 日本語店名（例: `居酒屋`）が `encodeURIComponent` でエンコードされた URL が `href` に設定されることを確認する
